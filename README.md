# MlabsDesafio

## Project dependencies
- Node - 14.17.4
- Npm  - 6.14.14

## Installation

For install de application its very simple.

```
git clone https://github.com/LuigiJordanio/mlabs-desafio
cd mlabs-desafio
npm install
```

## Start serve
To start the local server of your machine with the project just run the command

```
npm start
```
